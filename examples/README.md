# Examples

This directory contains example projects demonstrating Embedded32 platform capabilities.

## Directory Structure

```
examples/
├── 01-hello-can/              # Basic CAN send/receive
├── 02-j1939-monitor/          # J1939 message monitoring
├── 03-can-to-mqtt-bridge/     # Bridge CAN to MQTT
├── 04-raspberry-pi-gateway/   # Raspberry Pi gateway
├── 05-stm32-engine-sim/       # STM32 engine simulator
└── 06-diagnostic-tool/        # J1939 diagnostic tool
```

## Getting Started

Each example includes:
- README with setup instructions
- Complete source code
- Configuration files
- Hardware requirements

Browse the directories above to explore specific examples.
