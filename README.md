# Embedded32Bit_CAN
Embedded 32 bit CAN model to make simulation for car communication include Engine , Brake , (Electronic Control Units, or ECUs) with 32-bit (or sometimes 16-bit or 64-bit, depending on the application) microcontrollers designed for specific tasks like engine management, anti-lock braking systems (ABS), airbags, climate control, and infotainment.
